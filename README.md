# Project Name : Retinal Blindness (Diabetic Retinopathy) detection   
   
# Retinal Blindness Detection

Deep learning-based diabetic retinopathy detection using PyTorch, achieving ~93% accuracy.

Implemented as part of internship at Webmyne Systems Pvt. Ltd.

## Features
- Trained on 10,000+ retinal images
- Achieved 93% test accuracy
- GUI built using Tkinter for healthcare professionals
- Patient data managed via HeidiSQL

## Technologies
- PyTorch
- Tkinter
- HeidiSQL

## Acknowledgements
